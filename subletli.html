Subletli

A simple, user-friendly platform for posting and finding short-term rentals

Live Demo

Overview

Subletli is a web application designed to simplify the process of finding and posting short-term rental properties. Whether you're a student looking for a temporary place to stay or someone with an extra room to rent, Subletli provides an intuitive platform to connect renters and property owners.

Key Features

  • User Authentication: Secure login and registration system
  • Property Listings: Easy-to-use interface for posting rental properties
  • Search & Filter: Advanced search functionality with location and price filters
  • Responsive Design: Optimized for both desktop and mobile devices
  • Real-time Updates: Live property availability and pricing updates

Technology Stack

React JavaScript HTML/CSS Supabase Vercel

Challenges & Solutions

Challenge: Creating an intuitive user interface that works seamlessly across different devices and screen sizes.

Solution: Implemented a responsive design using CSS Grid and Flexbox, ensuring consistent user experience across all platforms. Used React's component-based architecture to create reusable UI elements.

Results

Subletli successfully provides a streamlined platform for short-term rentals, with a clean and intuitive user interface that makes property searching and posting effortless for users of all technical backgrounds.
